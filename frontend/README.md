# ServiceHub Frontend

Frontend React + TypeScript para o ServiceHub Diaristas.

## Requisitos
- Node.js 18+
- npm

## Como rodar
```bash
npm install
npm run dev
```

## Configuração da API
Crie um arquivo `.env` dentro de `frontend` com:
```
VITE_API_URL=http://localhost:8080
```

## Rotas
- `/login`
- `/cadastro`
- `/diaristas`
- `/diaristas/:id`
- `/historico`
